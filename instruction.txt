# Инструкция по использованию программы Video Audio Replacer

1. **Запуск программы**
   - Убедитесь, что Python 3.7 или выше установлен на вашем компьютере.
   - Установите зависимости, запустив команду:
     ```
     pip install -r requirements.txt
     ```
   - Запустите программу командой:
     ```
     python replace_audio.py
     ```

2. **Выбор папок**
   - В первом окне выберите папку, где находятся видеофайлы (форматы: `.mp4`, `.mov`, `.avi`).
   - Во втором окне выберите папку с аудиофайлами (форматы: `.mp3`, `.wav`, `.aac`).
   - В третьем окне выберите папку, куда будут сохранены измененные видео.

3. **Процесс работы**
   - Программа автоматически заменит звук во всех видеофайлах из первой папки, выбирая случайное аудио из второй папки.
   - Длина аудиофайла будет приведена к длине видео.

4. **Результат**
   - Обработанные видео будут сохранены в указанной выходной папке с префиксом `new_`.

5. **Завершение**
   - После завершения работы программы в консоли появится сообщение: `Готово!`.

# Примечания
- Убедитесь, что в выбранных папках есть файлы подходящих форматов.
- Если программа завершилась с ошибкой, проверьте, что пути к папкам корректны и файлы поддерживаемы.
